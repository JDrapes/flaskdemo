What is my project?

This project is titled learning patterns of activty using facial recognition and mobile phone broadcasts; this involves a classifier system attempting to classify MAC addresses to an individuals face over time. The project operates from a raspberry PI4 with a webcam and an alpha network card attached. The system is pre-trained on some faces, when it detects a face it recognizes it will scan the network and store the data. When there is enough the data the classifier will attempt to associate MAC addresses to the recognized faces. Below represents how this data would ideally be presented.
